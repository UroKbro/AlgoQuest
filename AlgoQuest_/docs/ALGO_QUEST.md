# ALGO_QUEST.md: The Universal Logic Ecosystem

**Project Name**: AlgoQuest  
**The Umbrella**: A unified "Algorithm Operating System" (AOS) that guides users from raw logic (The Dojo) to deep understanding (The Laboratory), complex experimentation (The Sandbox), and finally to real-world impact (The World).
**The Hook**: "Learn the code, master the logic, experiment with the complex, build the world."

---

## 1. The Four Realms (Visual Core)

### I. The Dojo (Foundational Mastery)
*Focus: Mastering Python through "Kinetic Syntax".*
- **The Engine**: **In-Browser Execution**. Uses **Pyodide** (Python in WebAssembly) to run code entirely in the user's browser. No local installation required.
- **The Loop**: Drop in code → AI reviews instantly → Provides feedback → Visualizes memory/logic → Repeat.
- **Visual Lessons**: Every lesson uses **Physical Metaphors** (Memory Boxes, Pointer Drones).

### II. The Laboratory (Multi-Dimensional Study)
*Focus: Deep diagnostic analysis of foundation algorithms.*
- **Features**: **The Telemetry Dock** (Live stats), **Logic Snapshotting** (Comparison), **The "What If?" Scrubber** (Time-travel).
- **Visual Clarity**: **Elastic Motion** and **Visual Diffing** for high-impact feedback.

### III. The Sandbox (Experimental Chaos)
*Focus: Breaking, stress-testing, and emergent play.*
- **Features**: **The Chaos Panel** (Parameter overload/Fault injector), **Structural Sabotage** (Deleting code mid-execution).
- **Visuals**: Optimized for 100,000+ data points using HTML5 Canvas.

### IV. The World (Project AI Loop)
*Focus: Applying logic to functional projects via AI partnership.*
- **The Flow**: Describe an idea → AI generates the code skeleton → User refines → AI provides critique/feedback → Visual updates.
- **The Bridge**: **Project Blueprints** provide pre-written skeletons for real tools. The **Deployment Bridge** allows for one-click browser-to-file export of finished projects.
- **Zero-Install**: No local environment needed. Everything happens in the AlgoQuest browser environment.

---

## 2. Integrated Interaction & AI
- **Socratic Tutor**: AI-driven "Discovery Anchors" that glow when the user hovers over a visual element, providing a 1-sentence explanation of the *visual change*.
- **Neural Bridge**: AI "Idea-to-Syntax" translator.
- **Iterative Feedback**: A tight loop where the AI reviews Dojo/World code and suggests logical refinements based on visual state.

---

## 3. Design System: "Cyber-Minimalism"
- **Aesthetic**: **Glass-on-Black**. Deep charcoal (`#050505`), frosted glass, and neon-pulse glows.
- **Palette**: **Logic Cyan** (Lab), **Growth Amber** (Dojo), **Emergent Purple** (Sandbox), **Project Emerald** (World).
- **Rule of 3**: No view requires >3 interactions to get from "Idea" to "Visual."

---

## 4. Technical Blueprint (Browser-First)
- **Frontend**: React.js + Pure CSS.
- **Execution Engine**: **Pyodide** (In-browser Python).
- **Boot Sequence**: A cinematic "Installing Logic Engine" splash screen that plays during the initial ~15MB Pyodide/Library download.
- **AI Feedback Model**: **Limited On-Demand Critique**. AI only analyzes code when the user clicks the "Review Logic" button, rather than streaming advice, ensuring focused learning and API efficiency.
- **User Persistence**: 
    - **Guest Mode**: All progress and code saved to `localStorage` (Device-specific).
    - **Cloud Sync**: Optional login to unlock **Weekly Stats**, **The Path (Analytics)**, and cross-device persistence.
