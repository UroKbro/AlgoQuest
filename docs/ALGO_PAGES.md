# ALGO_PAGES.md: The Universal Logic Map

This document serves as the structural blueprint for the 8 core pages of AlgoQuest. All pages follow the **Cyber-Minimalist** aesthetic: Glassmorphic cards, `#050505` backgrounds, and realm-specific neon accents.

---

## Page 1: The Nexus (Central Hub)
*The user's starting point and a high-level overview of their journey.*

- **Hero Section**: A "Quest Continuity" card. If a user was mid-lesson in the Dojo or mid-project in the World, this card displays a mini-visualizer of their last state with a "Resume Quest" button.
- **Mastery Pulse (Weekly Stats)**: A clean line graph showing "Active Minutes" and "Logic Problems Solved" for the current week. Below this, a "Weekly Focus" tag generated by the **Diagnostics Engine** (e.g., "Focus: Recursive Depth").
- **Realms Navigation**: Four large, high-impact Glass Cards (Dojo, Lab, Sandbox, World) that glow in their respective colors (Amber, Cyan, Purple, Emerald) on hover.
- **Interactions**: 
    - Clicking a Realm Card triggers a "Zoom-In" spring animation into that module.
    - Clicking the Mastery Pulse opens **The Path (Page 7)** for deeper analytics.

---

## Page 2: The Dojo (Foundations)
*The structured learning environment for Python and CS principles.*

- **Lesson Rail**: A vertical sidebar showing the "Path to Black Belt." Lessons are grouped by level (Core, Libraries, Architect).
- **The Stage**: A large central area where **Memory Boxes** and **Pointer Drones** visualize the code.
- **Integrated IDE**: A minimalist code editor at the bottom.
- **The Weekly Gate (Quiz)**: Every Sunday, the lesson content is replaced by "The Gate."
    - **Interaction**: Users must solve 3 logic puzzles and 2 code snippets. 
    - **The Feedback Loop**: If a user fails a "Loop" question, the Dojo automatically highlights and moves "Loop Mastery" lessons to the top of their rail for the next week.
- **Interactions**: Successfully passing a lesson triggers a "Transfer to Lab" button to see the principle applied in a full algorithm.

---

## Page 3: The Laboratory (Foundational Visualization)
*Deep study of classic algorithms (Sorting, Searching, Graphs).*

- **Visual Canvas**: A 60FPS HTML5 Canvas displaying the algorithm.
- **The Time-Travel Scrubber**: A tactile bar at the bottom with **Discovery Anchors** (white dots) that reveal 1-sentence logic tips on hover.
- **Live-Logic Sandbox**: A small, floating code-card on the right. 
- **Interactions**:
    - Changing code in the Sandbox triggers **Visual Diffing** (affected nodes pulse).
    - Scrubbing the timeline updates the Sandbox code highlight to the corresponding line.

---

## Page 4: The Sandbox (Complex Exploration)
*A playground for emergent and high-level logic.*

- **Complexity Canvas**: Handles 100,000+ data points (Slime Mold, Boids, Raft).
- **The Glass Console**: A right-side panel with etched sliders to adjust parameters (e.g., speed, repulsion, failure rate).
- **The Chaos Toggle**: A physical-looking switch that introduces random failures into the system.
- **Interactions**: Moving a slider creates an immediate "Ripple Effect" on the canvas with zero latency.

---

## Page 5: The World (Application)
*The space for real-world project building.*

- **Project Blueprint**: A "Skeleton" view of a functional project (e.g., a mini-search engine).
- **Implementation IDE**: A full-screen code editor that alternates with a "System Architecture" map.
- **The Feedback Loop**: A "Review Logic" button. 
    - **Interaction**: AI reviews the code and "ghosts" the architectural map to show where data flow is inefficient.
- **Interactions**: Projects completed here can be exported as "Logic Posters" to **The Forge**.

---

## Page 6: The Forge (Creative Social)
*A minimalist gallery of community creations.*

- **Posters Gallery**: A masonry grid of "Logic Posters" (snapshots of Lab or Sandbox states).
- **Challenge Browser**: A list of user-generated inputs (e.g., a "Nightmare Maze" for pathfinding).
- **Interactions**: Clicking a challenge instantly launches it into **The Laboratory** with those exact parameters.

---

## Page 7: The Path (Analytics & Portfolio)
*The deep-dive into user growth and verified skill.*

- **Mastery Radar**: The 5-axis SVG spider graph (Logic, Syntax, Efficiency, Projects, Speed).
- **Weekly Breakdown**: A detailed table of your **Weekly Gate** results, identifying "Strengths" and "Friction Points."
- **Badge Vault**: A 3D shelf of earned Belts and Medals.
- **Interactions**: Hovering over a "Friction Point" provides a "Teleport to Dojo" button to fix that specific inconsistency.

---

## Page 8: The Terminal (Settings & Ecosystem)
*The control center for the AOS.*

- **Ecosystem Hub**: Toggle the Browser Extension settings.
- **Personalization**: Sliders for neon intensity, sound effect volume, and motion blur.
- **Technical Sync**: Clear local cache or export all progress as a JSON file.
- **Interactions**: Changing neon intensity updates the site-wide CSS variables in real-time.
