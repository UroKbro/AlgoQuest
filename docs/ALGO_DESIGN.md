# ALGO_DESIGN.md: Cyber-Minimalism Design Spec

AlgoQuest’s design focuses on removing friction and making logic "feel" tangible through depth, light, and motion.

---

## 1. Universal Visual Clarity (The "No-Manual" Rule)
Logic must be understood through sight before text. 

- **Variables**: Floating 3D "Memory Boxes."
- **Loops**: Circular/Linear tracks where a drone moves step-by-step.
- **The "AI Critique" UI**: Non-intrusive floating glass panels that highlight specific code lines and visual elements simultaneously.

---

## 2. The Realm Identity System
Each realm uses a distinct color energy:
- **The Dojo (Warm Amber - `#F59E0B`)**: Learning and wisdom.
- **The Laboratory (Logic Cyan - `#00F2FF`)**: Diagnostic study.
- **The Sandbox (Ultraviolet - `#A855F7`)**: Complex/Emergent play.
- **The World (Project Emerald - `#10B981`)**: Functional software.

---

## 3. Interaction Principles

1.  **Zero-Install Loop**: The UI should never ask the user to leave the browser. Everything from coding to AI feedback happens in a single, fluid window.
2.  **Single-Player Focus**: Removed all social/multiplayer clutter to ensure a deep, focused "Discovery" experience.
3.  **The Rule of 3**: No view requires more than 3 interactions to move between "Idea" and "Visual."
4.  **Spring-Loaded UI**: Every panel opening uses **Spring physics** (stiffness: 120, damping: 20) for a weighted feel.

---

## 4. Visual Feedback
- **Impact Zone**: When the AI suggests a change, the affected visual elements "pulse" in the realm's accent color.
- **Success Emerald**: A full-screen pulse for correctly implemented logic or solved challenges.
